# Sanity 初始化清单

在开始使用网站之前，您需要在 Sanity Studio 中创建以下初始数据：

## 1. 启动 Sanity Studio

```bash
npm run sanity
```

访问 http://localhost:3333 打开 Sanity Studio

## 2. 必须创建的初始数据

### 2.1 CTA設定 (ctaConfig)

**必须创建至少一个 CTA 配置文档**

1. 在 Sanity Studio 中，点击左侧菜单的 "CTA設定"
2. 点击 "Create new" 创建新文档
3. 填写以下字段：
   - **プライマリCTA**
     - テキスト: `資料請求`
     - リンク: `/資料請求`
   - **セカンダリCTA**
     - テキスト: `無料相談`
     - リンク: `/無料相談`
4. 点击 "Publish" 发布

**注意**: 如果没有 CTA 配置，Hero 区域和侧边栏的 CTA 按钮将不会显示。

### 2.2 業種カテゴリ (industryCategory)

**建议创建以下业种分类**（参考 ai-souken.com）：

1. 在 Sanity Studio 中，点击左侧菜单的 "業種カテゴリ"
2. 为每个业种创建文档，填写：
   - **値 (value)**: 用于 URL 和筛选的唯一标识（英文，如 `general`, `construction`, `it` 等）
   - **表示名 (displayName)**: 显示给用户的名称（如 `全般`, `建設・建築`, `IT・システム開発` 等）
   - **説明 (description)**: 可选
   - **並び順 (order)**: 数字，用于排序（越小越靠前）

**建议的业种列表**：
- 全般 (value: `general`, order: 1)
- 建設・建築 (value: `construction`, order: 2)
- 人材 (value: `hr`, order: 3)
- IT・システム開発 (value: `it`, order: 4)
- 不動産 (value: `real-estate`, order: 5)
- 医療・福祉・介護 (value: `healthcare`, order: 6)
- 金融・保険 (value: `finance`, order: 7)
- 飲食・食品 (value: `food`, order: 8)
- 教育 (value: `education`, order: 9)
- サービス (value: `service`, order: 10)
- 製造 (value: `manufacturing`, order: 11)
- 広告 (value: `advertising`, order: 12)
- 自治体・公的機関 (value: `government`, order: 13)
- 研究・製薬 (value: `pharma`, order: 14)
- その他 (value: `other`, order: 15)

**重要**: 创建事例（Case）时，必须选择一个业种分类。如果没有业种分类，事例列表页的筛选功能将无法正常工作。

### 2.3 事例 (Case)

创建示例事例：

1. 点击 "事例" 菜单
2. 创建新文档，填写：
   - **タイトル**: 事例标题
   - **スラッグ**: 自动生成（基于标题）
   - **課題分類**: 选择一个課題（单选）
   - **業種カテゴリ**: 选择一个业种（必须）
   - **技術タグ**: 选择一个或多个技术标签
   - **抜粋**: 简短摘要
   - **本文**: 详细内容
   - **特徴画像**: 可选
   - **公開日**: 发布日期
   - **おすすめ事例**: 勾选后会在首页显示

### 2.4 メソッド (Method)

创建示例メソッド：

1. 点击 "メソッド" 菜单
2. 创建新文档，填写：
   - **タイトル**: メソッド标题
   - **スラッグ**: 自动生成
   - **技術タグ**: 至少选择一个（必填）
   - **抜粋**: 简短摘要
   - **本文**: 详细内容
   - **特徴画像**: 可选
   - **公開日**: 发布日期
   - **おすすめメソッド**: 勾选后会在首页显示

### 2.5 サービス (Service)

创建服务页面：

1. 点击 "サービス" 菜单
2. 创建新文档，填写：
   - **タイトル**: 服务名称
   - **スラッグ**: 自动生成
   - **抜粋**: 简短描述
   - **本文**: 详细说明
   - **特徴画像**: 可选
   - **技術タグ**: 可选
   - **公開日**: 发布日期

## 3. 验证清单

在开始使用网站之前，请确认：

- [ ] 已创建至少一个 CTA 配置文档
- [ ] 已创建至少一个业种分类（用于事例筛选）
- [ ] 已创建至少一个事例（用于测试）
- [ ] 已创建至少一个メソッド（用于测试）
- [ ] 已创建至少一个サービス（用于测试）

## 4. 常见问题

### Q: 事例列表页显示错误 "Unable to parse value of $industry=undefined"
**A**: 已修复。确保在创建事例时选择了业种分类。

### Q: 首页没有显示精选内容
**A**: 确保在创建事例或メソッド时勾选了 "おすすめ事例" 或 "おすすめメソッド" 选项。

### Q: CTA 按钮不显示
**A**: 确保已创建 CTA 配置文档并已发布。

### Q: 筛选功能不工作
**A**: 确保已创建业种分类，并且事例文档中已选择业种。
